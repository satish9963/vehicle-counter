❌ TROUBLESHOOTING

Problem: "Build Failed" (red ✗)
Solution:
1. Click on the failed workflow
2. Scroll down to see error message
3. 95% of time: Missing files
4. Make sure you uploaded ALL 39 files
5. Try again

Problem: No "Actions" tab
Solution:
1. Refresh the page
2. Wait 30 seconds
3. GitHub might need time to detect workflow
4. Try again

Problem: Can't find "Upload files" option
Solution:
1. Click green "[<> Code]" button
2. Look for dropdown arrow (▼)
3. Select "Upload files" from menu
4. Drag files into upload area

Problem: Uploaded files but build didn't start
Solution:
1. Make sure build-exe.yml was uploaded
2. Check it's in the right location
3. It should be in root folder, NOT in subfolder
4. Refresh page and check Actions tab
