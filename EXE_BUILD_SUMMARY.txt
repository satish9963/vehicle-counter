TROUBLESHOOTING QUICK REFERENCE

═══════════════════════════════════════════════════════════════════════════

Can't find Python?
→ Reinstall from python.org
→ CHECK: "Add Python to PATH"
→ Restart Command Prompt

Can't find Node?
→ Reinstall from nodejs.org
→ Use default settings
→ Restart Command Prompt

setup.bat fails?
→ Check internet connection
→ Free up disk space (need 50GB)
→ Run as Administrator
→ Try again

build.bat fails?
→ Disable antivirus temporarily
→ Free up disk space (need 15GB)
→ Verify setup.bat completed
→ Try again

Can't find dist folder?
→ Build didn't complete
→ Check last error message
→ Most likely: disk space or antivirus
