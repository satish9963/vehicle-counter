# 🚀 QUICK BUILD GUIDE - Windows .EXE in 3 Steps

## 3-STEP PROCESS

### ✅ Step 1: One-Time Setup (10-15 minutes)

**Download & Install**:
1. **Python** → https://www.python.org/downloads/
   - Choose 64-bit installer
   - ✅ **IMPORTANT**: Check "Add Python to PATH"

2. **Node.js** → https://nodejs.org/
   - Choose "LTS" version
   - Default settings OK

3. **Verify Installation**:
```bash
python --version    # Should show 3.9 or higher
node --version      # Should show v16 or higher
npm --version       # Should show 8.0 or higher
```

---

### ✅ Step 2: Prepare Project (5 minutes)

**In Command Prompt**:
```bash
# Navigate to project folder
cd C:\path\to\vehicle-counter

# Run setup
setup.bat

# Wait for completion
# You'll see: "[SUCCESS] Setup Complete!"
```

**What it does**:
- Creates Python virtual environment
- Installs all Python packages
- Downloads YOLOv8 model (160MB)
- Installs Node.js packages
- Creates necessary folders

---

### ✅ Step 3: Build .EXE (5-10 minutes)

**In Command Prompt**:
```bash
# Run builder
build-windows-exe.bat

# Wait for completion
# You'll see: "[SUCCESS] BUILD SUCCESSFUL!"
```

**What gets created**:
- `dist\Vehicle Counter Setup 1.0.0.exe` (Installer - 500MB)
- `dist\Vehicle Counter 1.0.0.exe` (Portable - 400MB)

## ⚙️ TROUBLESHOOTING

### "Python not found"
```
1. Reinstall Python
2. ✓ Check "Add Python to PATH"
3. Restart Command Prompt
```

### "Node not found"
```
1. Reinstall Node.js
2. Use default settings
3. Restart Command Prompt
```

### "setup.bat failed"
```
1. Check internet connection
2. Ensure 50GB free disk space
3. Run as Administrator
4. Try again
```

### "build-windows-exe.bat failed"
```
1. Disable antivirus temporarily
2. Ensure 15GB free disk space
3. Check that setup completed
4. Try again
```

### "dist folder not created"
```
1. Check error message in console
2. Make sure build reached "[SUCCESS]"
3. Check C:\vehicle-counter\dist\ exists
4. If not, re-run build script
```

## 📞 QUICK HELP

**Problem**: "Missing module X"
**Fix**: Delete venv folder, re-run setup.bat

**Problem**: "Port 8000 already in use"
**Fix**: Close other Vehicle Counter windows, try again

**Problem**: "Takes too long to build"
**Fix**: Normal! Takes 5-10 minutes. Don't close window.

**Problem**: ".exe files too large"
**Fix**: Normal! Includes Python, Node, all dependencies. ~400-500MB is expected.

**Problem**: "Can't find dist folder"
**Fix**: Look in C:\vehicle-counter\dist\
If not there, build didn't complete successfully.
