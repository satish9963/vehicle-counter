SECTION 7: TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

PROBLEM: "Python not found"
SOLUTION:
  1. Reinstall Python from python.org
  2. ✓ Check "Add Python to PATH"
  3. Restart Command Prompt

PROBLEM: "Node not found"
SOLUTION:
  1. Reinstall Node.js from nodejs.org
  2. Use default settings
  3. Restart Command Prompt

PROBLEM: setup.bat fails
SOLUTION:
  1. Check internet connection
  2. Make sure 50GB free disk space
  3. Run Command Prompt as Administrator:
     - Right-click cmd → "Run as administrator"
  4. Try again

PROBLEM: build-windows-exe.bat fails
SOLUTION:
  1. Disable antivirus temporarily
  2. Ensure 15GB free disk space
  3. Verify setup.bat completed successfully
  4. Try again

PROBLEM: App won't start
SOLUTION:
  1. Check if port 8000 is available:
     > netstat -ano | findstr :8000
  2. If there's a process using it, close that process
  3. Try running app again

PROBLEM: Can't find dist folder
SOLUTION:
  1. Check: C:\vehicle-counter\dist\
  2. If it doesn't exist, build didn't complete
  3. Look at last message in Command Prompt
  4. Most likely: Not enough disk space or antivirus blocked
